Robot Control Panel

Control the movement of a robot by adjusting the angles of its motors using sliders on the settings.

Real-Time Interaction with Pygame and Robot

Control a virtual 2D character using Pygame and see the corresponding movements in a connected physical robot.

User-Friendly Interface

Enjoy a simple and intuitive control interface for an engaging user experience.

Customizable

Easily adapt the code to fit your own robot's specifications, making it suitable for various hardware configurations.

How to Use

  1. Clone the repository to your local machine.
  2. Install the required dependencies using pip install -r requirements.txt.
  3. Change the server address to the IP address of the robot.
  4. Start the Server in the server folder.
  5. Start the game.py file.
  6. Watch as the robot mirrors the movements of the virtual character in real-time!
